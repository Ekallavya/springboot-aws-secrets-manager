## Create AWS RDS Database
	-> Login AWS management console

	-> create AWS RDS DB ,ensure db is Available 
		
		ex: cloudtech-database-1
		schema/database : testdb
		

## Create AWS secretManager

	-> login AWS management console
	-> search SecretManager ,click on it
	-> navigate to Secrets
	-> click on Store a new secretManager
	-> select secret type : Credentials for Amazon RDS database (radio button)
	
		 other availabe options
		 
			Credentials for Amazon DocumentDB database
			Credentials for Amazon Redshift data warehouse
			Credentials for other database
			Managed external secret - Secrets vended by your third-party software vendor.
			Other type of secret -API key, OAuth token, other.
			
	-> navigate to credentials
		give input user name ( actual db username )
		give input password  ( actual db password )
		
	-> default encription key :aws/secretsmanager
	-> navigate to Database
	-> select DB instance
	   ex: cloudtech-database-1
	-> click on Next
	-> navigate to Configure secret bydefault
	->  secret name :give input here
		
		ex: cloudtech-rdsdb-secrets-dev
	-> click on next
	-> navigate to Configure rotation bydefault
	-> click on next
	-> navitgate to review bydefault	
	-> click on store
	-> after succesful creation ,verify Secrets list for confirmation
